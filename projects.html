Weather Research Application

Course Project - Illinois State University - IT 326 Spring 2025

A comprehensive command line application designed for meteorological research and analysis. The application provides real-time weather data and storm logs by integrating multiple APIs including NOAA and Nominatim. Features include radar visualization, weather-focused logbook entries, and comprehensive storm tracking capabilities that support both amateur and professional meteorological research. Developed alongside three peers to satisfy the requirements of IT 326 at Illinois State University, with a focus on practicing software engineering principles and best practices.

Key Technologies: Java, API Integration, Databases, Real-time Processing, Software Engineering Principles

Key Features:

  • Real-time weather data from multiple sources
  • Historical weather data analysis and trending
  • Storm tracking and severe weather alerts
  • Export capabilities for research data
